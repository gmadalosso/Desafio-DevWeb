Sobre o projeto

Contexto

O projeto "SOS Saúde RS" foi desenvolvido para a disciplina Desenvolvimento para Web do curso de Análise e Desenvolvimento de Sistema da Universidade do Vale dos Sinos (Unisinos).

Sob o tema proposto "Cuidado com a Saúde" e sob o contexto das enchentes que afetavam o estado do Rio Grande do Sul durante o mês de maio de 2024, decidiu-se criar um portal para divulgação de informações sobre saúde pertinentes ao contexto dos afetados pelas enchentes, bem como um formulário de triagem que, de forma hipotética, pudesse ajudar profissionais da saúde a avaliar e encaminhar pacientes ao tratamento adequado de forma mais rápida e eficiente.

Tecnologias utilizadas

O projeto foi inteiramente desenvolvido utilizando as tecnologias de desenvolvimento front-end HTML, CSS e JavaScript. Também foi utilizado o framework BootStrap 5.
